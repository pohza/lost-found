# User Manual - Lost & Found (Frontend)

## 1. การเข้าสู่ระบบ (Login)

1. เปิดหน้าเว็บ (เช่น `http://localhost:5173` หรือ URL ที่ deploy จริง)
2. กดปุ่ม **"Log in"** (ถ้าอยู่หน้าสมัคร) หรือเข้าที่ `/login`
3. กรอก **Email** และ **Password**
4. กดปุ่ม **Continue**
5. ถ้าข้อมูลถูกต้อง ระบบจะพาไปหน้า **Home**

> ถ้า login ไม่สำเร็จ จะมีข้อความแจ้งเตือนใต้หัวข้อ Login

## 2. การสมัครสมาชิก (Signup)

1. ที่มุมขวาบนของหน้า Login กด **"Sign Up"**
2. กรอกข้อมูล:
   - Email or Phone
   - Password
   - Full Name
3. กดปุ่ม **Continue**
4. ถ้าสมัครสำเร็จ ระบบจะ login ให้อัตโนมัติ และพาไปหน้า Home

## 3. หน้า Home – ค้นหาประกาศของหาย/ของพบ

### ฟังก์ชันหลัก
- แถบค้นหา (Search) สำหรับค้นหาด้วย keyword
- ตัวกรอง (Filter): สถานที่, วันที่เริ่มต้น/สิ้นสุด
- การจัดเรียง (Sort): ล่าสุด / ใกล้วันพบ / ยอดเข้าชม
- แท็บ **Lost Items** / **Found Items** สำหรับสลับประเภทประกาศ

### วิธีใช้งาน
1. เลือกแท็บด้านบน (Lost / Found)
2. ใส่คำค้น หรือเลือก filter ที่ต้องการ
3. เลือกค่าใน dropdown Sort
4. เลือกรายการเพื่อดูรายละเอียดเพิ่มเติม (Item Detail)

## 4. การลงประกาศของหาย (Report Lost)

1. ไปที่เมนู **REPORT** แล้วเลือก **Lost**
2. กรอกข้อมูลในฟอร์ม:
   - Item Name
   - Location
   - Date & Time ที่ของหาย
   - Description (รายละเอียดเพิ่มเติม)
   - รูปภาพ (ไม่บังคับ)
3. กดปุ่ม **Submit** เพื่อส่งประกาศ
4. ถ้าสำเร็จ ประกาศของคุณจะไปอยู่ในหน้า Home และ My Posts

## 5. การลงประกาศของที่พบ (Report Found)

1. ไปที่เมนู **REPORT** แล้วเลือก **Found**
2. กรอกข้อมูล:
   - Item Name
   - Location
   - Date & Time ที่พบ
   - Description
   - วิธีส่งมอบ (handover):
     - ส่งให้จุดรับของกลาง
     - เก็บไว้เองชั่วคราว
   - วิธีติดต่อเพื่อรับคืน (how to claim)
   - รูปภาพ (ถ้ามี)
3. กดปุ่ม **Submit**

## 6. รายละเอียดประกาศ (Item Detail)

เมื่อคลิกที่ประกาศจากหน้า Home:
- จะแสดง:
  - รูปภาพ
  - ชื่อของ
  - เวลา/วันที่พบหรือหาย
  - สถานที่พบหรือหาย
  - Description

### การส่งคำขอรับคืน (Submit Claim)
- ถ้าคุณไม่ใช่เจ้าของประกาศ (isOwner = false) จะเห็นปุ่ม **Submit Claim**
- กดปุ่มแล้วระบบจะพาไปหน้า Claim ของรายการนั้น

### เมนูของเจ้าของประกาศ
- ถ้าคุณเป็นเจ้าของ (isOwner = true) จะเห็นปุ่ม:
  - แก้ไข (ไปที่หน้า Report Lost/Found ในโหมดแก้ไข)
  - ประกาศของฉัน (ไปที่ My Posts)

## 7. การส่งคำขอรับคืน (Claim)

1. ในหน้า Claim กรอกข้อมูล:
   - Full Name
   - Contact Email
   - Contact Number
   - Description (อธิบายหลักฐานความเป็นเจ้าของ)
   - Serial Number (ถ้ามี)
   - รูปหลักฐาน (optional)
2. กดปุ่ม **Submit Claim**
3. ถ้าสำเร็จ ระบบจะแสดงข้อความยืนยัน และ backend จะใช้ข้อมูลนี้ในการตัดสินใจ

## 8. การดูสถานะคำขอรับคืน (Claims)

### Claims Submitted
- แสดงรายการคำขอที่คุณส่งไปแล้ว
- แสดงสถานะ เช่น Submitted / Need More Info / Completed / Canceled

### Claims Status
- รวมรายการ claims ทั้งหมดที่เกี่ยวข้องกับคุณ (ทั้งที่เป็นผู้ส่งและเจ้าของโพสต์ ตามการออกแบบ backend)

## 9. การจัดการประกาศของฉัน (My Posts)

1. ไปที่เมนู **My Posts**
2. จะเห็นรายการประกาศที่คุณเป็นคนลง
3. แต่ละรายการสามารถ:
   - แก้ไข (Edit) → เปิดหน้า Report Lost/Found แบบโหลดข้อมูลเดิม
   - ปิดประกาศ (Close) → เมื่อของถูกส่งมอบคืนแล้ว
   - ลบประกาศ (Delete)

## 10. การดูแจ้งเตือน (Notifications)

1. ไปที่เมนู **NOTIFICATION**
2. หน้า Notifications จะแสดงรายการแจ้งเตือน เช่น มีคนส่ง Claim ใหม่
3. คลิกที่รายการเพื่อดูรายละเอียดในหน้า Notification Detail

ในหน้า **Notification Detail**:
- แสดงข้อมูลผู้ส่ง claim: ชื่อ, อีเมล, เบอร์โทร, Description
- เจ้าของประกาศสามารถกด:
  - **Approved** (อนุมัติ)
  - **Canceled** (ยกเลิก)

## 11. ข้อความในระบบ (Messages)

### Messages (รายการ)
- ไปที่เมนู **MESSAGES**
- เห็นรายการ thread การสนทนาเกี่ยวกับประกาศ/claim ต่าง ๆ
- แต่ละรายการแสดงหัวข้อ, ชื่อประกาศ, และจำนวนข้อความใหม่ (unread)

### Message Detail
- เมื่อคลิกที่ thread ใด thread หนึ่ง:
  - แสดงประวัติข้อความในหัวข้อนั้น
  - คุณสามารถพิมพ์ข้อความในกล่องข้อความด้านล่าง แล้วกด **ส่ง**

## 12. โปรไฟล์ผู้ใช้ (Profile)

1. ไปที่เมนู **PROFILE**
2. แก้ไขข้อมูล:
   - Full Name
   - Email
3. ระบบจะบันทึกผ่าน API `/api/me`
4. ปุ่ม **Log out** จะล้าง token ใน localStorage และพากลับไปหน้า Login
